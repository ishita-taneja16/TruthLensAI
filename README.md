# 🕵️ TruthLensAI – RAG-Enabled Generative AI Lie Detection Platform with Agentic Workflows

> An AI-powered lie detection and interview intelligence system that leverages Retrieval-Augmented Generation (RAG), multi-agent orchestration, and semantic memory to conduct adaptive interviews. The system retrieves relevant conversational context using vector search and generates intelligent follow-up questions using Gemini for behavioral consistency analysis.

---

# 🚀 Overview

TruthLensAI is an intelligent interview analysis platform that dynamically adapts interview questions based on user responses instead of relying on predefined questionnaires. By combining RAG, LangGraph-based agent orchestration, FAISS vector search, and Gemini LLM, the system retrieves relevant conversational memory, detects inconsistencies, and performs contextual behavioral reasoning to improve interview reliability.

# ⚙️ System Workflow

1. User starts an interview session.
2. Facial verification authenticates the participant.
3. Heart-rate baseline is established.
4. User responses are converted into embeddings.
5. FAISS retrieves relevant conversational context (RAG).
6. LangGraph orchestrates multiple AI agents.
7. Gemini generates context-aware follow-up questions.
8. Behavioral consistency and deception indicators are evaluated.
9. Results are displayed on the dashboard.
